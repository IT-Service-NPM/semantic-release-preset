# @it-service-npm/semantic-release-preset Semantic-release shareable configuration

A shareable [semantic-release](https://github.com/semantic-release/semantic-release)
configuration,
for enforcing consistent GitHub/NPM releases.

## Purpose

This configuration also includes a semantic-release configuration,
which enables automated GitHub/NPM releases based on commit messages.
